# Healthy Heartrate Breathing

Forked from the Reachy Mini conversation app.

Use the `src/healthy_heartrate_breathing/profiles/_healthy_heartrate_breathing_locked_profile` folder to customize your own app from this template:
- Edit instructions `_healthy_heartrate_breathing_locked_profile/instructions.txt`
- Edit available tools in `_healthy_heartrate_breathing_locked_profile/tools.txt`
- You can create your own tools in `_healthy_heartrate_breathing_locked_profile` by subclassing the `Tool` class.

Do not forget to customize:
- this `README.md` file
- the `index.html` file (Hugging Face Spaces landing page)
- the `src/healthy_heartrate_breathing/static/index.html` (the web app parameters page)

The original README from the conversation app is available in `README_OLD.md`.
